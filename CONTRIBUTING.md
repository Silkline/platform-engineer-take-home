# Contributing

This repository is a take-home exercise for interview candidates, not an actively maintained project.

If you are a candidate:

1. Fork this repo to your personal GitHub account.
2. Work on a branch in your fork.
3. Open a PR from your branch to your fork's `main`.
4. Send the PR URL via the email thread you received the take-home from.

See [README.md](./README.md) for the full prompt.

External pull requests opened against this upstream repository will be closed without review — the exercise is the work, not the patch.
