# Security

**This repository contains deliberately insecure infrastructure code.**

It is a take-home exercise for candidates interviewing for Silkline's Founding Infrastructure / Platform Engineer role. The Pulumi code in `index.ts` includes intentional security and operational mistakes — finding and fixing them is the exercise.

**Please do not:**

- Deploy this code to a real AWS account.
- File security advisories or CVEs against this repository. The issues are intentional.
- Treat any "credentials," tokens, or secrets in this repo as real. They are placeholders.

If you spotted something we *didn't* plant and want to give us a heads-up, the best channel is the email thread you received the take-home from.
